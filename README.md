# SweetCrumb 🍰

SweetCrumb is a responsive bakery e-commerce website built using HTML, CSS, and JavaScript.

## Features
- Product listing with categories
- Add to Cart functionality
- Dynamic cart updates using localStorage
- Checkout page with OTP simulation
- Order tracking page
- Responsive UI design
- Video banner integration

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- localStorage API
